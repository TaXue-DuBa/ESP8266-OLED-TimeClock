# Esp8266-oled-timeclock
ESP8288 OLED 128x64 SSD1306 timeclock

其中`MyRealTimeClock`为第三方库，Arduino中的库管理器没有，，所以就上传上来了

下载后将此文件夹放到`C:\Users\用户名称\Documents\Arduino\libraries`文件夹中，
